# Stock Market Analytics Dashboard

A Python + Streamlit web app for stock analyis, dividend income tracking, and portfolio analytics.

---

## Features

### Stock Analysis
- Price charts (line + candlestick)
- Moving Averages (MA20, MA50)
- Daily returns distribution
- Drawdown analysis
- Benchmark comparison (e.g., vs SPY)

### Dividend Tracking
- Dividend yield + annual per share
- Estimated annual & monthly income
- Dividend history table
- Monthly dividend breakdown (income by month)

### Portfolio Mode
- Multi-stock portfolio input
- Market value + allocation %
- Annual + monthly dividend income by stock
- Income concentration analysis
- Top dividend contributor insights
- Portfolio allocation chart

### Data Export
- Download stock data (CSV)
- Download dividend history (CSV)
- Download monthly dividend breakdwon (CSV)
- Download full portfolio breakdown (CSV)

---
